# DeepSeek Desktop

Forked from [doxdk/deepseek-desktop](https://github.com/doxdk/deepseek-desktop)

## Added Features

- **Prompt Presets** — Save up to 9 frequently-used prompts with `{clipboard}` placeholder. Manage via the ⚙ button (top-right).
- **Keyboard Shortcuts** — `Alt+C` toggles the window; `Alt+1`–`Alt+9` apply saved presets.
- **Menu Integration** — Shortcuts are visible in the **Window** and **Presets** menus.
- **Auto-focus Textarea** — When the window gains focus, the cursor automatically moves to the chat input.
- **Bug Fix** — Removed broken state-restoration logic that caused infinite page reloads.
